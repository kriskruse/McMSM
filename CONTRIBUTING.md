# Contributing

Thanks for contributing to McMSM.

## License of contributions

By submitting a contribution (pull request, commit, or patch) to this repository, you agree that:

- Your contribution is provided under `AGPL-3.0-only` for inclusion in this project.
- You have the right to submit the contribution.
- The project maintainer may use your contribution in dual-licensed distributions of McMSM (including commercial licenses).

If this does not work for you, please do not submit the contribution.

## Development flow

1. Fork the repository.
2. Create a branch for your change.
3. Open a pull request with a clear description and rationale.
4. Be responsive to review feedback.
